# Shoe-Sales-Dashboard
Interactive Shoe Sales Dashboard built using Looker Studio to analyze sales performance, profit, product categories, transactions, and regional trends through dynamic visualizations and filters.
PROJECT OUTCOME: 
This dashboard helps businesses monitor sales performance, identify profitable regions and products, and make informed decisions through interactive visualizations and real-time filtering capabilities.
The Shoe Sales Dashboard is an interactive business intelligence dashboard developed using Looker Studio to analyze shoe sales performance across different states, cities, product categories, and order types. The dashboard provides key business metrics, interactive filters, and visual insights to help stakeholders monitor sales trends and make data-driven decisions.
PROJECT OBJECTIVES :
* Analyze overall shoe sales performance.
* Track sales amount, cost, quantity, and transactions.
* Compare sales across states and cities.
* Evaluate product category performance.
* Monitor order types and customer-level sales.
* Provide an interactive dashboard for business analysis.
DASHBOARD FEATURES :
* Sales Amount KPI
* Cost Analysis
* Quantity Sold
* Total Transactions
* Maximum Sales Value
* State & City Filters
* Product Category Filter
* Customer Name Search
* Order Type Analysis
* Interactive Google Maps Visualization
* Product Category Distribution
* Monthly Sales Trend
* State-wise Profit Analysis
TOOLS USED :
Google Looker Studio
Google Sheets / CSV Dataset
KEY INSIGHTS :
* Identified top-performing states based on sales and profit.
* Compared sales performance across different shoe categories.
* Analyzed customer purchasing patterns and transaction volume.
* Monitored monthly sales trends to identify business performance.
* Enabled dynamic filtering for better decision-making.
